Menu Items

1-Pile Strategies:

  • Cautious (CTRL+C) Computer player will take only 1 stick each round.
  • Greedy (CTRL+E) Computer player will leave only 1 stick for the human player to take - and ultimately lose the game.
  • Random (CTRL+R) Computer player will take a random stick each round. This round is the most unbiased and least predictable.

At any point, the user can close the window or end the game by selecting Menu >> Game >> Exit (CTRL-X).
