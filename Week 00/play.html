Let's Play!

Rules

  1. Once the game loads, user will be prompted with Nim rules. Select yes or no for this alert to load at the beginning of each game.
  2. User can set the initial pile size to 9, 16, or 21
  3. User can select either the 'human first' or 'better computer first' or 'random player' which will randomly select either the human or computer to take the first turn
  4. When it's the human's turn, the player can select between 1 and 3 (Max) sticks to take
  5. Select Take Turn
  6. The computer will then take its turn
  7. **NOTE: By default, the computer's strategy is 'Cautious'. Only one stick will be taken. To have the computer play a different strategy, select the strategy menu. See menu for more information on the various strategies the computer can play.
  8. The player who takes the last stick has lost the game
  9. If you'd like to play again, select Game >> New (CTRL+N).
