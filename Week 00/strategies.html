Nim Strategy

How to win:

  • Try to develop a winning strategy from the first move. The winning strategy is always applied to the player who's able to position the number of stones on the board to equal Nim-Sum of 0 (XOR = 0).

Calculating Nim-Sum:

  • Distinct Powers of 2: To find Nim-Sum, convert the number of stones in each pile to distinct powers of 2 (example: multiples of 4, 2, and 1). Cancel pairs with equal multiples and add the remaining numbers.
  • XOR: Nim-Sum can also be calculated using Exclusively-OR. Convert each number to binary and add the binary values.
  • If the result for either of these methods is 0, Nim-Sum = 0 and this is considered a winning strategy. If the result does not equal 0, this is considered a losing strategy.

Example:

  • The starting piles contain stone counts of 1, 3, 5, and 7. Using the distinct powers of 2 method, this can be broken down to
    1. 1
    2. 2 - 1
    3. 4 - 1
    4. 4 - 2 - 1
    where the one set of 4, one set of 2, and two sets of 1 cancel each other out. This indicates the first player has a losing strategy since the Nim-Sum = 0. The objective is to ensure each time a stone is removed from the pile, Nim-Sum = 0. The only exception is when playing Misère. For Misère, your last move should ensure that Nim-Sum does NOT equal 0 so you can retrieve the final stone.
