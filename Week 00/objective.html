Nim Objective & Rules

Rules

  1. Game must be played between 2 players
  2. There can be any number of heaps (piles) consisting of any number of objects (stones)
  3. Each player must remove one or more stones from a pile
  4. During a single turn, stones cannot be removed from different piles
  5. The game is over when there are no stones left on the board
