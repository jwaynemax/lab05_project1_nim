Nim User Guide

The following is a user guide detailing the history, rules, and variations of the game Nim. Nim is a combinational game played between 2 players. There is a series of piles (the number of piles does not matter) each containing several objects, often referred to as stones. During each players turn, they're required to take at least one or more stones from a single pile. Multiple stones cannot be taken from different piles in a single turn. A user is allowed to remove all the stones from a single pile in one turn. The game ends when there are no stones remaining on the board.
